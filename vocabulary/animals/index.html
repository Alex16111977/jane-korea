📖 Грамматика: Говорим о животных

🐾 Счётные слова для животных:

마리 - счётное слово для животных

• 개 한 마리 - одна собака

• 고양이 두 마리 - две кошки

• 새 세 마리 - три птицы

🏠 Где живут животные:

животное + 이/가 + место + 에서 + 살아요

• 사자가 아프리카에서 살아요 - Львы живут в Африке

• 개가 집에서 살아요 - Собаки живут дома

🍖 Что едят животные:

животное + 이/가 + еда + 을/를 + 먹어요

• 토끼가 당근을 먹어요 - Кролики едят морковь

• 고양이가 생선을 먹어요 - Кошки едят рыбу

🔊 Звуки животных:

животное + 이/가 + звук + 해요

• 개가 멍멍해요 - Собака лает (гав-гав)

• 고양이가 야옹해요 - Кошка мяукает

🏃 Действия животных:

Полезные глаголы:

• 날다 - летать (새가 날아요)

• 헤엄치다 - плавать (물고기가 헤엄쳐요)

• 뛰다 - бегать (말이 뛰어요)
